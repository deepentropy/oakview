This project is to be used as a javascript Library in other projects.

We want to stricly mirror the TradingView web interface, pixel perfect.

You develop in TypeSCript

You use lightweightcharts v5 API only (not v4):

* doc: https://tradingview.github.io/lightweight-charts/docs
* tutorials: https://tradingview.github.io/lightweight-charts/tutorials
* api reference: https://tradingview.github.io/lightweight-charts/docs/api
* as you were trained on v4, make sure to refer to the migration guide to v5: https://tradingview.github.io/lightweight-charts/docs/migrations/from-v4-to-v5

You understand the plugin creation: https://tradingview.github.io/lightweight-charts/docs/plugins/intro

And the indicators integration: https://tradingview.github.io/lightweight-charts/tutorials/analysis-indicators
